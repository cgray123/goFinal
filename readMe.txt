Start program by running go run main.go in the backend folder as while as running npm start in the frontend folder
Open as many copies of http://localhost:3000/ for the amount of users you want, all user will automatically be added to the Default chatroom 
enter names for your user in the first input box
enter /help in the input box for all commands (a few are listed below)
/make#'chatroomName' makes a new chatroom, /join#'chatroomName' adds that user to the chatroom 'chatroomName' can be anything
there are 3 different types of messages: enter a message into the input box will send that message to all chatrooms the user is in
use /dc_userName:msg will send a massage just to the user you entered, and  use /chat#chatroomName:msg will send a message only to the entered chatroom
